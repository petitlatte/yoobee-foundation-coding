JSON data Structure

JSON Stringify

A common use of JSON is to exchange data to/from a web server. When sending data to a web server, the data has to be a string. Convert a JavaScript object into a string with JSON.stringify(). 

								    	
								    		
								    		// We’ll look at a JSON object that we assign to the 
								    		// variable obj, and then we’ll convert it using 
								    		// JSON.stringify() by passing obj to the function. 
								    		// We can assign this string to the variable 's' 

								    		var obj = {"first_name" : "Sammy", "last_name" : "Shark", "location" : "Ocean"}

								    		var s = JSON.stringify(obj)

								    		// Now, if we work with s, we’ll have the JSON available to 
								    		// us as a string rather than an object. 

								    		'{"first_name" : "Sammy", "last_name" : "Shark", "location" : "Ocean"}'

JSON Parse

Strings are useful for transporting but you’ll want to be able to convert them back to a JSON object on the client and/or the server side. While you can convert text to an object with the eval() function, it is not very secure, so we’ll use the JSON.parse() function instead.
