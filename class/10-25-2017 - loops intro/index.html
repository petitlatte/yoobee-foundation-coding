For Loops

As long as the condition within the loop is true, the curly brackett code block will continue to run.

Notice how the i variable within the condition itself is set, tested then incremented.

Duckett online example 
										
											var scores = [24, 32, 17];      // Array of scores
											var arrayLength = scores.length;// Items in array
											var roundNumber = 0;            // Current round
											var msg = '';                   // Message
											// Loop through the items in the array
											for (var i = 0; i < arrayLength; i++) {
											  //Try using a console.log within the loop to visualize it:
												console.log('a');
											  // Arrays are zero based (so 0 is round 1)
											  // Add 1 to the current round
											  roundNumber = (i + 1);
											  // Write the current round to message
											  msg += 'Round ' + roundNumber + ': ';
											  // Get the score from the scores array
											  msg += scores[i] + '
';
											}
											document.getElementById('answer').innerHTML = msg;

While Loops

We use the previous for loops when the number of iterations are known. We use while loops when the number of iterations are unknown.

Simple while loop syntax:

										
											// When the number of iterations are not known
											while( i < 10)
											{
											    //Some code goes here
											}
Duckett online example 
										
											var i = 1;       // Set counter to 1
											var msg = '';    // Message

											// Store 5 times table in a variable
											while (i < 10) {
											  msg += i + ' x 5 = ' + (i * 5) + '
';
											  i++;
											}

											document.getElementById('answer').innerHTML = msg;

Do While Loops

Example at w3

This loop will always be executed at least once, even if the condition is false, because the code block is executed before the condition is tested:

										
											var text = "";
											var i = 0;
											do {
											    text += "The number is " + i;
											    i++;
											}
											while (i < 5);

											var table = 3;             // Unit of table

											var i = 1;                 // Set counter to 1
											var msg = '';              // Message

											  }

';
											    i++;
											  }
											}

											// Write the message into the page
											var el = document.getElementById('blackboard');
											el.innerHTML = msg;

setTimeout() / setInterval()

setTimeout()

										
										// window.setTimeout(function, milliseconds);

										var x = document.getElementById("txt");
										setTimeout(function(){ x.value = "2 seconds" }, 2000);
										setTimeout(function(){ x.value = "4 seconds" }, 4000);
										setTimeout(function(){ x.value = "6 seconds" }, 6000); 

										//or
										setTimeout(function(){ alert("Hello"); }, 3000);

The window.setTimeout() method can be written without the window prefix. The first parameter is a function to be executed. The second parameter indicates the number of milliseconds before execution.

setInterval()

The setInterval() method repeats a given function at every given time-interval. The window.setInterval() method can be written without the window prefix. The first parameter is the function to be executed. The second parameter indicates the length of the time-interval between each execution. This example executes a function called "myTimer" once every second (like a digital watch).

										
										// window.setInterval(function, milliseconds);

										//or
										setInterval(function(){ alert("Hello"); }, 3000);
										
										//or
										//Here we can see it invoking the function myTimer which
										//is defined further down the script. (after 1000 miliseconds)

										//setInterval() is being assigned to a variable so that we can clear
										// the interval via the same variable with clearInterval()
										var myVar = setInterval(function(){ myTimer() }, 1000);

										function myTimer() {
										    var d = new Date();
										    var t = d.toLocaleTimeString();
										    document.getElementById("demo").innerHTML = t;
										}
